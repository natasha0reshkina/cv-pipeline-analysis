# Исследовательский проект: пайплайн компьютерного зрения

Данный репозиторий содержит исследовательский прототип, разрабатываемый
в рамках курсового проекта.
Цель проекта анализ компромиссов между производительностью, качеством
и устойчивостью в пайплайнах компьютерного зрения.

## Описание проекта

В рамках проекта рассматривается программный пайплайн компьютерного зрения,
включающий загрузку изображений, предобработку, выполнение модели и
постобработку результатов.
Основная задача исследования изучить влияние параметров обработки изображений
и параметров исполнения пайплайна на задержку обработки, пропускную способность,
ресурсоёмкость, метрики качества и устойчивость к типовым искажениям входных данных.


## Текущее состояние

На момент контрольной точки 2:
- выбран базовый сценарий компьютерного зрения
- подготовлена структура репозитория проекта
- начата реализация основных этапов пайплайна
- закладывается инфраструктура для воспроизводимых экспериментов
  с варьированием параметров обработки и исполнения

## Планируемые работы

- расширение реализации пайплайна и параметров конфигурации
- добавление логирования метрик и профилирования этапов пайплайна
- проведение серии экспериментальных прогонов
- анализ компромиссов скорость, качество, устойчивость
- выявление рисков эксплуатации и формирование практических рекомендаций

## Структура репозитория

- `src/` — реализация пайплайна
- `experiments/` — скрипты для запуска экспериментов
- `configs/` — конфигурационные файлы экспериментов
- `requirements.txt` — зависимости проекта
