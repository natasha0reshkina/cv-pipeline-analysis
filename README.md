# Анализ производительности, качества и рисков в пайплайнах компьютерного зрения

В работе я анализирую, как параметры обработки изображений, свойства самих данных и добавление типовых искажений влияют на время обработки, качество результата и устойчивость пайплайна. Основная исследовательская часть собрана в ноутбуке `курсовая.ipynb`, где последовательно выполняются подготовка данных, запуск экспериментов, расчет агрегированных метрик и построение графиков.

## О чем проект

В центре работы находится тестовый пайплайн обработки изображений. Для него я сравниваю несколько наборов данных, отличающихся по сложности и разрешению, а также отдельно проверяю поведение системы на изображениях с шумом, размытием и jpeg сжатием

В ходе экспериментов оцениваются:
- задержка обработки
- верхний квантиль времени обработки
- разброс измерений
- качество преобразования изображения по метрикам PSNR и SSIM
- устойчивость пайплайна на искаженных данных

## Что находится в репозитории

`курсовая.ipynb` содержит основную исследовательскую часть проекта

`data` содержит описание используемых наборов данных и пояснение, где хранятся изображения

`docs` содержит текстовые материалы по протоколу эксперимента, воспроизводимости и рискам

`results` содержит сырые результаты прогонов, агрегированные таблицы и сохраненные графики

`src` содержит вспомогательные модули для работы с пайплайном, измерениями и файлами

`configs` содержит основные параметры экспериментов

`scripts` содержит заготовки для возможного переноса части логики из ноутбука в отдельные сценарии запуска

`reports` содержит итоговый текстовый отчет по проекту

## Что было сделано в работе

В рамках проекта я:
- собрала и подготовила несколько наборов изображений для разных сценариев исследования
- реализовала тестовый пайплайн обработки одного изображения
- провела серию измерений на разных размерах входного изображения
- рассчитала метрики производительности и качества
- отдельно проанализировала разброс измерений и самые медленные случаи
- проверила устойчивость пайплайна на данных с искусственными искажениями
- сформулировала ограничения исследования, риски и практические рекомендации

## Основные результаты

Проведенные эксперименты показали, что увеличение размера входного изображения улучшает качество преобразования, но одновременно увеличивает задержку обработки. Анализ разброса измерений показал, что стабильность работы зависит не только от выбранной конфигурации, но и от свойств самих изображений. Отдельная проверка на искаженных данных показала, что наиболее сильное влияние на устойчивость по времени обработки оказывает шум

## Как использовать репозиторий

Основной файл проекта ноутбук `курсовая.ipynb`. Именно в нем собраны подготовка данных, запуск экспериментов и анализ результатов. Остальные папки дополняют ноутбук и содержат описание данных, результаты запусков и сопроводительные материалы
