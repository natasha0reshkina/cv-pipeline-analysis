# Результаты

В этой папке собраны результаты экспериментов, полученные в ходе работы с ноутбуком `курсовая.ipynb`. Здесь находятся сырые таблицы отдельных прогонов, сводные таблицы с агрегированными метриками, а также сохраненные графики. Большое количество файлов связано с тем, что эксперименты запускались несколько раз на разных наборах данных и при разных значениях входных параметров

Файлы `bench` содержат исходные результаты отдельных запусков. Файлы `agg_summary.csv`, `agg_perf.csv` и `agg_quality.csv` используются как основные итоговые таблицы для анализа производительности и качества. Файл `agg_quality_psnr.csv` отражает более ранний этап работы, когда качество оценивалось только по PSNR. Файл `tail_top20.csv` нужен для анализа самых медленных изображений, а папка `figures` содержит построенные по результатам графики

Часть файлов в папке относится к промежуточным и повторным прогонам, поэтому для итоговых выводов используются, прежде всего, агрегированные таблицы и визуализации
