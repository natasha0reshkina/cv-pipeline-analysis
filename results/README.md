# Результаты

В этой папке собраны таблицы и артефакты, полученные в ходе экспериментов из ноутбука `курсовая.ipynb`.

Основные файлы:
- `bench_*.csv` — сырые результаты отдельных прогонов для основных наборов данных и наборов с искажениями
- `agg_summary.csv` — агрегированные метрики производительности
- `agg_perf.csv` — объединенная таблица метрик производительности
- `agg_quality.csv` — агрегированные метрики качества по PSNR и SSIM
- `agg_quality_psnr.csv` — ранняя версия таблицы качества только с PSNR
- `tail_top20.csv` — самые медленные изображения по наборам данных
- `corr_summary.csv` — агрегированные результаты на искаженных изображениях
- `figures/` — сохраненные графики

Часть файлов отражает промежуточные прогоны и несколько запусков одного и того же эксперимента. Для итогового анализа используются агрегированные таблицы и визуализации.
